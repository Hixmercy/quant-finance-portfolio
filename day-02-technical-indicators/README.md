# Day 2 – Technical Indicator Dashboard (EURUSD)

## Objective
This project builds a simple dashboard to analyze market trends and momentum using technical indicators.

## What Was Done
- Calculated moving averages (20-day and 50-day)
- Computed RSI (Relative Strength Index)
- Built MACD (Moving Average Convergence Divergence)
- Visualized price trends and signals

## Key Insights (Simple Explanation)
- The market moves in trends, not randomly
- Indicators help identify when momentum is strong or weak
- No single indicator is enough — combining them gives better insight

## Why This Matters
This project shows how raw price data can be transformed into meaningful signals that traders can use to make decisions.

## Tools Used
- Python
- Pandas
- Matplotlib

Next, we will convert these indicators into actual trading strategies and test their performance.
